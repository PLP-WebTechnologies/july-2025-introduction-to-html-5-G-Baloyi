About This Page

This is a basic example of a web page using HTML5. It includes a clear structure, basic accessibility features, and beginner SEO practices.

Accessibility Basics

We use semantic elements like <header>, <nav>, and <main> to help screen readers understand the layout of the page. Navigation has an ARIA label to make it easier to find.

SEO Tips

Each section has a heading, and the page includes a title and meta description. These help search engines understand and display your page correctly.
